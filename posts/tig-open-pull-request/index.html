open-pr command

$ vim open-pr
$ chmod +x open-pr

ref: https://gist.github.com/mknkisk/381acbb8c87f67df5e3f

Browse Github pull request page

$ open-pr {commit id}

Custom bindings to tig

custom bindings to tig in your ~/.tigrc

+ bind generic P @open-pr %(commit)
